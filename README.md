# 📚 StudyTask Manager

## Giới thiệu
StudyTask Manager là ứng dụng quản lý lớp học thông minh, giúp giáo viên chủ nhiệm và học sinh theo dõi công việc, thời khóa biểu và tiến độ học tập.

## Tính năng

### 👨‍🏫 Giáo viên chủ nhiệm
- Quản lý thời khóa biểu
- Giao bài tập cho cả lớp
- Theo dõi tiến độ học sinh
- Gửi thông báo đến lớp
- Xem thống kê tổng quan

### 👩‍🎓 Học sinh
- Xem việc cần làm hôm nay
- Tick hoàn thành công việc
- Xem thời khóa biểu
- Nhận thông báo từ giáo viên
- Bảng xếp hạng lớp
